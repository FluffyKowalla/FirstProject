Brief HTML Reference

Tags and Description

Name

Tag

Description
Body <body> The body element represents the content of an HTML document. Tags in the body element render on the web browser.
Headings <h1> - <h6> The heading wlwments are ordered from highest to lowest with the font size and boldness becoming lower the higher the h(number).
Division <div> Divides the HMTL document into sections.
Paragraph <p> Contains a block of text.
Span <span> An inline container for text for styling (reframe from use if a more semantic element is available).
Emphasis <em> Allows you to emphasize text usually italicize
Strong <strong> Highlights selected text within tag in bold.



Attributes

Name

Attribute

Description
ID id="myIdentification" The (id) attridute allows us to uniquely differentiate between elements and helps in the design aspect of cascading style sheets. Added to the opening of the tag.
Name name="" Added to the opening of the tag. To configure the behavior of the element.
ALT alt="description of image" Placed within an image tag (<img>) to display if image could not be rendered and/or to help the visually impaired providing a description of the image.
